# Mint App
React interface for [Generic Mint Dapp](https://genericmint.netlify.app/) on Rinkeby.



## Development
To setup the development environment first clone the repo:
```bash
git clone https://github.com/0xArdy/generic-mint-dapp.git && cd generic-mint-dapp
```

### Local env
Install dependencies:
```bash
npm install
```

Start the app:
```bash
npm run start
```

The app should be live at [http://localhost:3000](http://localhost:3000/)
